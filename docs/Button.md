`Button` (component)
====================



Props
-----

### `children`

Pass text as child or other component

type: `node`


### `disabled`

Disable button

type: `bool`
defaultValue: `false`


### `isLoading`

Toggle loading state with `isLoading`.
Child component or child text will not show while loading.

type: `bool`
defaultValue: `false`


### `onPress`

Function to call when button is pressed

type: `func`


### `onPressDisabled`

type: `func`


### `style`

type: `custom`
defaultValue: `{}`


### `textStyle`

type: `custom`
defaultValue: `{}`


### `underlayColor`

type: `string`
defaultValue: `'#222'`
